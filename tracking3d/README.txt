################################################################################
Here are neural networks used to classify detector hits to tracks within a
particle collision detector.

################################################################################
generator

"This file contains scripts used to generate aspects of data, such as adding
a random third dimension to a two-dimensional set."

################################################################################
models

"A collection of trained neural networks that can be loaded in with the keras
package."

################################################################################
notebooks

"A collection of notebooks used to show how data was loaded in, how it was
cleaned, how it was used to train a model and how it was used to evaluate a
model."

################################################################################
sets

"Contains the data used to train and evaluate the neural networks."

################################################################################
tracker

"Tools used to extract and clean data. Tools used to train and evaluate models.
Tools used to visualize data and data frames."
